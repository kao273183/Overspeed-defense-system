# <zh-TW>

## 🎉 首次內測版本 (v1.0.0)

### 【別扣我 - 主動式防禦系統】
這是一款專為台灣駕駛設計的駕照防護盾，專注於「嚴重超速預警」。

### 🔥 核心特色：
1. **🛡️ 主動防禦：**
   - 專注高風險情境（如：速限+40km/h 扣牌風險）。
   - 可自定義「嚴重超速門檻」與「緩衝區」。
   - 僅在真正危險時發出強烈警報，告別無效干擾。

2. **🗺️ OSM 社群地圖整合：**
   - 採用 OpenStreetMap 即時偵測路段速限。
   - 不再依賴老舊固定照相資料。

3. **🔊 智慧背景執行：**
   - 支援背景運作，搭配 Google Maps 導航無衝突。
   - TTS 語音播報（例如：「嚴重超速！請煞車」）。

4. **🔧 缺漏回報與自我修正：**
   - 發現速限不準？一鍵標記並設定正確速限（Local Override）。
   - 支援上傳至 OSM 社群。

> ⚠️ **注意**：請務必開啟「永遠允許」位置權限以確保背景防護正常運作！

# <en-US>

## 🚀 First Internal Beta Release (v1.0.0)

### 【Don't Suspend Me - Active Defense System】
Your ultimate driver's license guardian, designed to prevent severe speeding penalties.

### 🔥 Key Features:
1. **🛡️ Active Defense:**
   - Focuses on high-risk scenarios (e.g., license suspension risks).
   - Customizable "Danger Threshold" and "Warning Buffer".
   - Alerts only when it matters, minimizing distractions.

2. **🗺️ OpenStreetMap Integration:**
   - Real-time speed limit detection using OSM data.
   - Smart dynamic alerts instead of static database reliance.

3. **🔊 Smart Background Service:**
   - Runs perfectly in the background with navigation apps.
   - Custom TTS Voice Alerts (e.g., "Severe Speeding! Brake Now!").

4. **🔧 Missing Data Fixes:**
   - Found a wrong limit? Mark it and set a Local Override instantly.
   - Upload accurate data to the OSM community.

> ⚠️ **Note**: Please enable "Always Allow" location permission for full background protection!
