🚗 GPS 智慧測速警示系統 (扣牌防禦版)
這是一個基於 Web 技術開發的純網頁版車用抬頭顯示器 (HUD) 替代方案。 專為台灣駕駛設計，針對「嚴重超速扣牌」法規（超速 40km/h 以上）提供最後一道防線的語音警示。無需安裝 App，打開瀏覽器即可使用。

✨ 核心特色
🛡️ 扣牌防禦邏輯：

系統預設在 「路段限速 + 38km/h」 時才會觸發警報。

範例：限速 110km/h，警報觸發點為 148km/h。

目的：容許正常超車或微幅超速，但確保不會觸發「嚴重超速 40 公里」的扣牌門檻。

🗺️ 自動路段限速 (OSM)：

整合 OpenStreetMap (Overpass API)，每 15 秒自動偵測當前座標的道路速限。

若地圖無資料，自動切換回手動設定值。

🔊 智慧語音播報：

使用瀏覽器原生 Web Speech API。

支援自定義警示語句（如：「嚴重超速！請立即煞車！」）。

支援藍牙車機播放（需將手機藍牙連接至車輛）。

👁️ 視覺化強烈警示：

正常行駛時顯示綠色時速。

觸發警報時，全螢幕紅底閃爍，利用眼角餘光即可察覺危險。

🔋 螢幕恆亮 (Wake Lock)：

透過 Screen Wake Lock API 防止手機在行駛中自動鎖定黑屏。

🛑 暫停監控模式：

進休息站或塞車時可一鍵暫停，停止 GPS 運作以省電。
